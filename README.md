# Simple NLP Chatbot (Python)

This is a simple rule-based NLP chatbot built using Python.  
It uses keyword matching to detect user intent and responds with predefined replies.

---
 Features

- Greeting detection (hello, hi, hey, etc.)
- Goodbye handling
- Thank-you responses
- Bot identity queries
- Text preprocessing (lowercasing and punctuation removal)
- Randomized responses for natural interaction

---

Technologies Used

- Python 3
- `re` (Regular Expressions)
- `random`
